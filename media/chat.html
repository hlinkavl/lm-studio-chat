~ lm-studio-chat
An agentic coding assistant that connects VS Code to your local LM Studio instance. The model has full read/write access to your workspace and can read files, propose edits, run shell commands, and call external tools via MCP — all without leaving the editor.
ask — every file write or command requires your Accept or Deny. Denied actions are remembered for the session.
edit — changes are applied immediately with a diff shown afterwards.
prompt — edit the system prompt sent to the model; workspace context and MCP tools are appended automatically.
shell & mcp — enable terminal access or connect MCP servers (databases, APIs) the model can call as tools.
